공모전 등록

배너 이미지

이 이미지가 공모전 상세 페이지 상단에 표시됩니다.

모든 기기에 최적화된 이미지가 표시되도록 2048x1152픽셀 이상, 6MB 이하의 이미지를 사용하세요.

공모전 제목

공모전을 잘 나타내는 제목을 입력하세요. 변경된 제목은 공모전 목록과 상세 페이지에 표시됩니다. 제목은 14일 내에 2회 변경할 수 있습니다.

주최자

공모전을 주최하는 기관 또는 이름을 입력하세요.

카테고리

공모전의 카테고리를 선택하세요.

접수 기간

공모전 참가 접수 시작일과 마감일을 설정하세요. 설정된 기간 동안만 참가 신청이 가능합니다.

~

결과 발표일

심사 완료 후 수상자를 발표할 날짜를 설정하세요. 발표일은 공모전 상세 페이지에 표시됩니다.

상금 정보

공모전 상금 정보를 입력하세요.

참가비

참가비가 있는 경우 금액을 입력하세요. 무료인 경우 비워두세요.

설명

태그

#을 붙여 입력한 뒤 Enter를 누르면 태그가 생성됩니다.
